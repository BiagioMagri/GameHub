🎮 GameHub | The Next-Gen Discovery Platform

GameHub è una Single Page Application (SPA) ad alte prestazioni dedicata all'esplorazione dell'universo videoludico. Sfruttando le API di RAWG, la piattaforma offre un'esperienza fluida, immersiva e visivamente audace, caratterizzata da un'estetica Neon-Dark moderna.

✨ Key Features
Esplorazione Dinamica: Navigazione avanzata tra migliaia di titoli filtrati per genere e popolarità.

User Experience Fluida: Integrazione di Lenis Scroll per uno scrolling inerziale morbido e transizioni eleganti.

Sistema di Ricerca & Paginazione: Motore di ricerca globale con gestione asincrona dei risultati e navigazione ottimizzata.

Area Personale: Autenticazione integrata per la gestione del profilo e salvataggio preferenze (in sviluppo).

Arcade Corner: Un tributo ai classici con il minigioco Snake integrato, sviluppato nativamente in React.

Design Responsive: Interfaccia ottimizzata per garantire la migliore esperienza su mobile, tablet e desktop.

🛠️ Tech Stack
Frontend Core: React.js + Vite.

Routing: React Router (utilizzando il pattern Loader per performance ottimali).

Styling: Tailwind CSS (Glassmorphism & Custom Animations).

Motion & UX: Framer Motion & Lenis Scroll.

API: RAWG Video Games Database.

🎨 Design Philosophy
L'interfaccia punta su un contrasto netto tra superfici scure (#0f1115) e accenti neon viola/fucsia. Ogni interazione è studiata per richiamare il feeling delle moderne dashboard gaming, con micro-interazioni costanti e feedback visivi immediati.
